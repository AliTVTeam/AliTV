loadKaryoFile(file, callback)

Load the karyo file and call the callback
Parameters:
Name Type Description
file string Path to the karyo.json file to load.
callback loadKaryoCallback The callback function to call after successfully loading the file, passes the content of the file as the first parameter.
Author:
  • Markus Ankenbrand
Source:
  • loadKaryo.js, line 12
Example
// calls an anonymous function, that logs the content of karyo.json to the console
loadKaryoFile('karyo.json', function(data){
	console.log(data);
});
