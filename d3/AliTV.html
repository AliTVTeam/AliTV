AliTV (Alignment Toolbox and Visualization) is a software for visualizing Whole Genome Alignments. It generates publication-ready figures which can be used for the easy comparision of complete genomes. AliTV stands out due to its interactive functions for analyzing the alignments directly onto the figure.

If you need help with AliTV, please checkout the documentation, website or the repository on GitHub.
